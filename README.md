# 2D Top-Down Adventure Game

A 2D top-down adventure game built with Unity and C#.

# Some bioms


## Features

### Player System

* 4-direction movement
* Rigidbody2D movement
* Walking animation
* Idle animation
* Health system
* Mouse-click attack system

### Enemy System

* Detection radius
* Chase AI
* Enemy health
* Enemy attack
* Enemy death
* Multiple enemy support

### Camera System

* Cinemachine camera follow
* Smooth camera movement

### World System

* Tilemap-based map
* Collision system
* Map exploration

## Controls

| Key               | Action     |
| ----------------- | ---------- |
| W                 | Move Up    |
| A                 | Move Left  |
| S                 | Move Down  |
| D                 | Move Right |
| Left Mouse Button | Attack     |
